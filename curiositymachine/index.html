Curiosity Machine

A web application that hosts STEM projects kids can engineer and connects them with professional mentors to help them learn. Iridescent hired me in 2014 to work full time on Curiosity Machine as a designer and developer. Since then, I've focused both on improving the existing interface and adding new features.

Because this project is ongoing and spans a long period of time, I've highlighted a few of the features I worked on below.

Go to curiositymachine.org →
  • Timeline: 2014 - Present
  • Project: Django web application
  • Role(s): Product design, front-end development
  • Collaborators: Cat Greim, Mike Larsson (and many others at Iridescent)
